# 子千

子千的初衷，是希望开发一个用于快速搭建网站前台，接口和中后台管理网站的代码模板项目。

目前包含以下模块：

- laravel：Laravel 后端接口项目
- vue3： 基于 Vitesse 魔改的空项目
- admin：基于 Vue3 的后台项目
- h5： 基于 Vue3 的 H5 的空项目
- uniapp：基于 uni-app 的空项目
- gateway：基于 Workerman Gateway 的空项目
- vitepress：Vitepress 空项目
- doc：子千文档
